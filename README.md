# Survive the Wasteland

Godot 3.2
=========

**A test project for Godot 3.2 game engine**

Features
========

- Looking around
- Walking/sprinting
- Jumping
- Sounds
- Weapon Spread
- Weapon Sway
- A map
- Ammunition and weapon pickups
- Basic AI

Keys
====

- Walk: `W/A/S/D`
- Sprint: `Left Shift`
- Jump: `Space`
- Fire: `Left Mouse Button`
- Reload: `R`
- Drop weapon: `G`
